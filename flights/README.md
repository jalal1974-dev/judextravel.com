# الجود للسياحة والسفر - Jude Travel Agency

## نظرة عامة | Overview

موقع إلكتروني حديث ومتجاوب لوكالة الجود للسياحة والسفر، متخصص في عرض رحلات الطيران المستأجرة من وإلى الأردن إلى 9 وجهات سياحية ودينية وتجارية مميزة.

A modern, responsive website for Jude Travel Agency showcasing charter flights from/to Jordan to 9 premium tourist destinations.

## 🔧 إصلاح تبديل اللغة | Language Toggle Fix

### ✅ **المشكلة التي تم حلها:**
كان زر تبديل اللغة يعمل بشكل معكوس - عند الضغط على "English" كان يُظهر العربية والعكس.

### 🛠️ **الحل المطبق:**
```javascript
// المنطق المُصحح
if (lang === 'ar') {
    // عند عرض المحتوى بالعربية
    currentLangSpan.textContent = 'English'; // الزر يُظهر ما يمكن التبديل إليه
} else {
    // عند عرض المحتوى بالإنجليزية  
    currentLangSpan.textContent = 'العربية'; // الزر يُظهر ما يمكن التبديل إليه
}
```

### 🎯 **السلوك الصحيح الآن:**
- **عند المحتوى العربي**: الزر يُظهر "English" (للتبديل للإنجليزية)
- **عند المحتوى الإنجليزي**: الزر يُظهر "العربية" (للتبديل للعربية)
- **التبديل الفوري**: تغيير اللغة والاتجاه بنقرة واحدة
- **حفظ تلقائي**: يتذكر اللغة المختارة عند إعادة زيارة الموقع

### 📱 **على جميع الأجهزة:**
- **سطح المكتب**: زر "🌐 English/العربية" في شريط التنقل
- **الجوال**: أيقونة الكرة الأرضية + خيار في القائمة المنسدلة
- **متسق**: نفس المنطق على جميع الشاشات والأجهزة
